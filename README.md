# Sherine CV Website (Static)

This is a lightweight, single-page personal website built from the provided CV.

## Files
- `index.html` — content and structure
- `styles.css` — styling
- `script.js` — search, accordion controls, mobile nav, mailto form

## How to run locally
Just open `index.html` in a browser.

If you prefer a local server:
- Python: `python3 -m http.server 8000`
- Then open: http://localhost:8000

## Deploy (quick options)
- GitHub Pages: push these files to a repo and enable Pages on the `main` branch.
- Netlify / Vercel: drag-and-drop the folder (static site).
